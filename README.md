# Data-Analysis-Practice-6-60-Excel-Advanced-Filter
This repository contains my Excel practice on Advanced Filter. In this project, I used the Advanced Filter feature to extract specific records from a dataset based on different conditions such as Month, Product, and Country. This practice is part of my Data Analysis Practice Series (6/60) to improve my Excel data analysis skills.

## Project Objective
The main objective of this project is to learn how Advanced Filter works in Excel to extract data using multiple conditions.
This practice helps to:
•	Understand Advanced Filter
•	Apply AND and OR conditions
•	Filter data more efficiently
•	Improve Excel data analysis skills

## Tasks Covered
Use Advanced Filter to show records where Month = Feb.
Use Advanced Filter to show records where Product = Shirt white.
Use Advanced Filter to show records where Country = USA.
Use Advanced Filter to show records where Month = Feb OR Month = Apr.
Use Advanced Filter to show records where Product = Shirt blue AND Country = USA.

## KPI (Key Performance Indicators)
•	Correct use of Advanced Filter
•	Correct application of AND condition
•	Correct application of OR condition
